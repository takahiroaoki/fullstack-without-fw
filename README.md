# fullstack-without-fw

A fullstack web application sample without frameworks.

## Requirements

- Docker Desktop
- VSCode & Dev Container Extension

※I use codespaces.

## How to boot

```
$ make setup
$ make run
```

Then, get access to `http://localhost:8080/XXX`
